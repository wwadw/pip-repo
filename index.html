How To Add A New Wheel

Keep the same folder pattern and this landing page stays easy to maintain.

  1. Create a folder: dist/new_pkg/
  2. Put the wheel file inside that folder.
  3. Add dist/new_pkg/index.html that points to the wheel file.
  4. Copy one package card above and change the 4 visible fields.

The root page stays readable because every package uses the same folder layout. 

dist/
  new_pkg/
    index.html
    new_pkg-0.1.0-py3-none-any.whl
